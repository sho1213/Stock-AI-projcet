## 1. エグゼクティブサマリー

### 結論

**ネットスターズの連結売上高のFY2026/12〜FY2030/12における平均年間オーガニック成長率（CAGR）は、ベースケースで約12.4%と推定する。**

年度別の成長率は、FY2026の+19%からFY2030の+9%へ漸減するパターンを描く。主因は(1)QRコード決済市場の成長率鈍化（年率24%→7%）と(2)テイクレート（決済手数料率）の構造的低下圧力である。DXソリューションの拡大はアップサイド要因だが、現時点で売上構成比9%にとどまり、全体成長率への寄与は限定的である。

会社のFY2026/12予想（売上高5,760百万円、前年比+20.3%）は近接年として概ね妥当と判断するが、5年間にわたり20%成長を維持できるとは考えにくい。

## 2. 事業概要と収益構造

### 事業概要

ネットスターズは2009年設立のフィンテック企業で、マルチキャッシュレス決済ソリューション「StarPay」を主力サービスとして提供する。StarPayは複数のQRコード決済ブランド（PayPay、d払い、楽天ペイ等）およびクレジットカード・電子マネーを単一のゲートウェイで加盟店に提供する決済インフラである。2023年9月に東証グロース市場へ上場した。

2026年1月にはエンターテインメント向けオンライン決済の新子会社「StarPay-Entertainment」を設立し、スマホ新法（2025年12月施行）を契機としたアプリ外課金市場への参入を開始している。

### 収益構成（FY2024/12期）

| セグメント | 構成比 | 内容 |
|-----------|--------|------|
| 決済手数料 | 82.4% | GPV×テイクレートで算出される決済仲介手数料 |
| 端末販売 | 2.6% | StarPay対応決済端末の販売 |
| DXソリューション | 9.2% | StarPay-DX（ミニアプリ等）の開発・提供 |
| その他 | 5.8% | 保守・コンサルティング等 |

売上の約85%が決済関連であり、**事業の根幹はGPV（決済取扱高）×テイクレート（平均決済手数料率）のモデル**である。

### 連結売上高推移

| FY | 売上高（百万円） | YoY | 備考 |
|----|-----------------|-----|------|
| 2019/12 | 1,390 | — | |

FY2024の+4.9%はミスリーディングである。2023年4月に通信事業を終了しており、FY2023には通信事業収入が部分的に含まれていた一方、FY2024にはゼロとなった。会社はFY2023の調整後成長率を+46.1%と開示しており、通信事業の年間売上は数億円規模であったと推定される。FY2025をクリーンなベースイヤーとして採用する。

## 3. KPI分析

### 3.1 GPV（決済取扱高）トレンド

| FY | GPV（億円） | YoY | 備考 |
|----|------------|-----|------|
| 2022/12 | ~9,000 | — | |
| 2023/12 | 13,166 | +46.3% | 1兆円突破 |
| 2024/12 | ~17,440 | ~+32% | 期初会社計画ベースの推計 |
| 2025/12 | ~21,000 | ~+20% | Q1-Q3実績15,312億から推計（Q4は季節的に最大） |

GPV成長率は**+46%→+32%→+20%と明確に減速トレンド**にある。これはQRコード決済市場全体の成長鈍化を反映している。ただし、ネットスターズのGPV成長率は市場全体（2024年：+23.9%）を上回る年もあり、シェア獲得力は一定程度認められる。

FY2025のGPV推計について：Q1-Q3累計が1兆5,312億円（前年同期比+35.7%）。Q4は年末商戦・忘年会需要により通常最大の四半期となることから、通期GPVを約2兆1,000億円と推計した。

### 3.2 テイクレート分析

テイクレートを「総売上高÷GPV」（オールインベース）と「決済手数料売上÷GPV」（決済ベース）の2通りで算出する。

| FY | 総売上/GPV | 決済売上/GPV（推計） | 備考 |
|----|-----------|-------------------|------|
| 2023/12 | 0.283% | ~0.235% | 通信事業売上を含む |
| 2024/12 | ~0.224% | ~0.184% | 通信事業除去後、ベースリセット |
| 2025/12 | ~0.228% | ~0.188% | 安定化の兆候 |

FY2023→FY2024の急低下は通信事業終了の一時要因が大きい。FY2024→FY2025では概ね横ばいであり、**通信事業除去後のテイクレートは0.22-0.23%近辺で安定化しつつある**。

ただし、構造的な低下圧力は継続する：
- **大口加盟店ミックスの変化**: SMCC経由・Stripe経由の大型加盟店はボリュームディスカウントが適用され、ブレンドレートを押し下げる
- **競争激化**: 決済代行業界は300社超が参入しており、手数料率引き下げ競争が常態化
- **1トランザクション当たりコスト**: 0.11円で低下傾向にあり、コスト面のオフセットは限定的

予測: FY2026 0.185% → FY2030 0.172%（決済ベース、年率約2%の漸減）

### 3.3 加盟店数・アカウント数推移

| 時期 | アカウント数 | 備考 |
|------|------------|------|
| 2020年 | ~20万拠点 | |
| 2022年 | ~30万拠点 | 1.3万社 |
| 2023年 | ~45万店 | 1.5万社 |
| 2024/12期末 | ~50万 | |
| 2025年（推定） | 60万超 | SMCC・Stripe連携効果 |

加盟店獲得の成長ドライバーは(1)セールスパートナー経由の直接獲得と(2)大手決済プレイヤーとのAPI連携によるゲートウェイ経由の間接獲得である。特にFY2024以降は三井住友カード（SMCC）がStarPayをQRコード決済ゲートウェイとして採用し、Stripeとの連携も進展したことで、獲得チャネルが拡大している。

ただし、加盟店数の増加がGPVや売上に比例するわけではない。大型パートナー経由の加盟店はテイクレートが低い傾向にあり、加盟店当たり売上（Revenue per Account）は漸減していると推定される。

## 4. 市場環境分析

### 日本のQRコード決済市場

|----|--------------------|----------|----------------------|
| 2023 | 10.9 | +37.5% | 39.3% |
| 2024 | 13.5 | +23.9% | 42.8% |
| 2025E | 16.4 | +21.5% | 47.2% |
| 2026E | 18.8 | +14.6% | 50.4% |
| 2027E | ~21.1 | ~12% | 52.7% |
| 2028E | ~23.2 | ~10% | 54.3% |
| 2029E | ~25.0 | ~8% | 55.5% |
| 2030E | ~26.8 | ~7% | 56.2% |

（出所：2024年実績は経済産業省、2025年以降予測はNRI「キャッシュレスビジョン」2025年発表を基に推計）

**最重要ポイント: QRコード決済市場の成長率は2024年の+24%から2030年には+7%程度へ大幅に減速する。** キャッシュレス比率の年間伸び幅も、2024→2025年の+3.7ptから2029→2030年の+0.7ptへ縮小する。市場はハイパーグロース期を終え、ステディグロース期に移行しつつある。

政府は2030年にキャッシュレス比率65%（国内指標）を中間目標として設定しており、最終的には80%を目指すとしている。この政策目標はポジティブだが、QRコード決済がその中でどの程度シェアを伸ばせるかは不確実である。現状ではクレジットカードが83%を占めており、QRコード決済のシェア（9.6%）が大幅に拡大するシナリオは楽観的と考える。

### ネットスターズへの示唆

ネットスターズの売上成長の天井はQRコード決済市場の成長率によって規定される。GPV成長率は「市場成長率＋シェア変動」であるが、市場成長率が7-10%に鈍化する環境下では、シェア拡大だけで20%超の成長を維持することは困難である。

## 5. 競争環境評価

### 主要競合との規模比較

| 企業 | GPV規模（推定） | 売上規模 | 特徴 |
|------|---------------|---------|------|
| GMOペイメントゲートウェイ | 25兆円超 | 売上収益~810億（年換算） | EC決済中心、圧倒的規模 |
| SBペイメントサービス | 10兆円目標(FY2025) | 非開示 | SBグループ、PayPayとの連携 |
| **ネットスターズ** | **~2.1兆円** | **47.9億円** | QR決済ゲートウェイ特化 |

ネットスターズの売上規模はGMO-PGの約1/17であり、決済代行業界においてはニッチプレイヤーの位置づけである。

### 競争優位性

- **QRコード決済の接続ブランド数国内最大級**（29ブランド）: 多ブランド対応のワンストップゲートウェイとしての差別化
- **システム可用性99.99%**: クラウドネイティブ・コンテナ技術による高い安定性
- **開発の柔軟性・スピード**: 新規決済ブランドへの迅速な対応力
- **大手パートナーからの採用実績**: SMCC、Stripe等が技術力を評価

### 競争上の課題

- **規模の劣位**: GMO-PGやSBPSと比較して圧倒的に小さく、価格交渉力で不利
- **製品の幅**: EC決済・クレジットカード決済ではGMO-PGに大幅に劣後。対面QR決済に偏重
- **人的リソース**: 連結従業員数223名（FY2024末）で、IPO時の233名からむしろ減少。売上が5年間で概ね倍増する場合、現行の人員体制で対応可能かは疑問。ゲートウェイ事業はスケーラブルだが、DX事業の拡大には人材が必要

### 評価

ネットスターズはQRコード決済ゲートウェイの先駆者として一定のポジションを確立しているが、中長期的には(1)市場のコモディティ化と(2)大手プレイヤーのQR領域参入により、競争優位性が徐々に希薄化するリスクがある。DXソリューションは差別化要因だが、現時点では売上構成比9%にとどまり、事業の柱とは言い難い。

## 6. 売上ドライバー分解と予測モデル

### モデル構造

```
連結売上高 = 決済手数料売上 + 端末販売 + DXソリューション + その他
         = (GPV × 決済テイクレート) + 非決済売上

GPV = QR決済市場規模 × ネットスターズ市場シェア
```

## 8. 感応度分析・リスク評価

### CAGR感応度マトリクス

GPV成長率（5年平均）とFY2030時点のテイクレートの組み合わせによるCAGR感応度：

| | テイクレート 0.160% | テイクレート 0.172% | テイクレート 0.185% |
|---|---|---|---|
| **GPV CAGR 10%** | ~7% | ~8% | ~10% |
| **GPV CAGR 13%（ベース）** | ~10% | ~12% | ~14% |
| **GPV CAGR 16%** | ~13% | ~15% | ~17% |

（DX他売上は年率16%成長で固定した場合の概算）

### 主要リスク要因（下振れ方向）

1. **テイクレート圧縮の加速**: 300社超の決済代行業者間の価格競争が激化し、大口加盟店の手数料率引き下げ要求が強まるリスク。ネットスターズの最大のバリュードライバーであるGPV成長が売上成長に直結しなくなる可能性
2. **QR決済市場の成長鈍化**: NRI予測よりも早期にキャッシュレス浸透が鈍化する場合。特に中高年層や地方での現金選好が根強く残るシナリオ
3. **大手競合のQR領域本格参入**: GMO-PGがEC領域からQRゲートウェイに進出した場合、規模・技術力・営業力で圧倒される可能性
4. **人的資源の制約**: 従業員223名（FY2024末）はIPO時の233名を下回る。DX事業や新規事業の成長には人材投資が必要であり、採用が追いつかない場合は成長のボトルネックとなる
5. **パートナー依存リスク**: SMCC、Stripe等の主要パートナーが自社ゲートウェイを構築した場合、GPV成長の前提が崩れる

### アップサイド要因（上振れ方向）

1. **パートナーシップの拡大**: SMCC・Stripe以外の大手カード会社・プラットフォーマーがStarPayを採用する場合
2. **DX売上の急加速**: StarPay-Entertainmentがスマホ新法を追い風にゲームプラットフォーム市場で成功する場合
3. **海外展開**: 現時点では子会社（大連、シンガポール、ベトナム）は小規模だが、東南アジアのQR決済市場（GrabPay、GoPay等）への本格進出が実現すれば新たな成長エンジンとなる
4. **クレジットカードゲートウェイの拡張**: QR決済に加えてクレカ・電子マネーのフルゲートウェイとしてのポジション確立
